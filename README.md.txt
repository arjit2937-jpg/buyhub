# CartZone

CartZone is a free, Amazon-inspired static e-commerce frontend template built entirely with HTML, CSS, and JavaScript. It includes a responsive design, modern homepage layout, product grid with images, pricing, and add-to-cart buttons (dummy functionality).

This template is ideal for learning web development, showcasing a sample online store, or experimenting with frontend design without backend integration. Fully compatible with GitHub Pages, it allows you to host and share a professional-looking shopping website in minutes.

## Features
- Clean, Amazon-style header with logo, search bar, and cart icon
- Responsive product grid layout
- Add-to-cart button with dummy alerts
- Mobile-friendly and fully responsive design
- Ready-to-use on GitHub Pages for free hosting

## How to Use
1. Upload all files to a **public GitHub repository**.
2. Enable **GitHub Pages** in repository settings (`Settings → Pages → main branch → /root`).
3. Open your GitHub Pages URL (e.g., `https://username.github.io/cartzone/`) in a browser.
